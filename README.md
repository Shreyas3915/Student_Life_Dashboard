🎓 Student Life AI Dashboard

An interactive AI-powered Student Analytics Dashboard that predicts student wellness, academic performance, and career readiness using machine learning-inspired scoring models.

The project combines **Python**, **HTML**, **CSS**, **JavaScript**, and **data preprocessing** to visualize student data through modern dashboards and generate actionable insights.

📌 Features

- 📊 Interactive Student Dashboard
- 🤖 AI-based Prediction Engine
- 📈 Burnout Risk Prediction
- 💼 Placement Readiness Score
- 📚 Backlog Risk Analysis
- 😊 Loneliness Detection
- ⚡ Productivity Score
- 📉 Weekly Productivity Trends
- 📡 Interactive Charts using Chart.js
- 👨‍🎓 Student Management System
- ➕ Add New Students
- 💾 Local Storage Support
- 📤 Export Dataset as JSON
- 📑 Dataset Visualization
- 🌙 Dark Mode Support
- 📱 Responsive Design

AI Prediction Metrics

The dashboard predicts five important student metrics.

| Metric | Description |
|---------|-------------|
| 🔥 Burnout Risk | Predicts mental exhaustion and stress level |
| 💼 Placement Score | Estimates placement readiness |
| 📚 Backlog Risk | Predicts possibility of academic backlogs |
| 😊 Loneliness Score | Measures social engagement and isolation |
| ⚡ Productivity Score | Predicts overall productivity |

Prediction Logic

The prediction engine evaluates student data using weighted scoring formulas to estimate:

- Burnout Risk
- Placement Score
- Productivity
- Loneliness
- Backlog Risk

The model combines academic performance, lifestyle habits, wellness indicators, and career-related activities to generate predictive scores.
